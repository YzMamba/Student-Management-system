# Student-Management-system
本学生管理系统是基于ssm框架并结合layui前端框架运用前后端异步交互完成，本系统分为三种用户，管理员负责专 业信息，班级信息，学生 信息、老师信息以及课程信息的录入修改删除等操作；学生选课 后老师可以查看班级学生并对学生课程进行打分，打分之后学生可以进入查看自己的学科成绩。
